Taking input(scanf)

Taking input is a very important aspect of programming. If you couldn't take input from the user
I wouldn't be able to write this to help you learn programming.

With that said there quite a bit of ways to get input from the user. The first one we're gonna talk about is scanf() .
scanf can take any kind of input. Numbers, strings, characters, floating point numbers, anything.
So let's see an example 



                    
#include 

int main(){
    int age;
    printf("What's your age? ");
    scanf("%d", &age);

    printf("%d\n", age);
                
}


This is a very simple program just initializing a variable called age. Then prompting the user to input their age.
And finally printing out their age.
But notice the &(ampersand) when we're using scanf(). To understand why that's there lets see how scanf() works.
scanf takes in the first characters/numbers up until to the first space. And it needs a placeholder to know what data type it's looking for.
%d/%s/%f/%c.
The &(ampersand) is used to store the value of the input into the variables memory location. And the memory location of a variable
is represented by the &(ampersand).
