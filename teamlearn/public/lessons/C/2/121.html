The If Statement

If statements are used to perform certain actions when a condition is met.
For example to know when someone's old enough to drink or something of that nature.
Let's see how translates into code 

                                
                                    #include <stdio.h>


                                    int main(){
                                        int age = 20;

                                        if(age => 18){
                                            printf("You are legally old enough to drink");
                                        }

                                        printf("You are too young to drink");

                                        return 0;
                                    }
