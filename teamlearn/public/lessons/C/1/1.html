First Chapter of the C course

To start off, we first need to make a "Hello World" example!

    
    #include 

    int main()
    {
      printf("Hello World!");

      return 0;
    }


Which will generate:
Hello World!

Breaking it down

To understand what is going on, we must break it down into pieces.

To start off, is the first step. It imports a keyword from the C standard library.

    
    int main()
    {
    
    }

This is the main method, where all the main code goes into. Note the fact that it has two curly braces. These are what we will use for all function bodies.

Then, inside the main function, we add this:

printf("Hello world!");

printf prints somthing to the screen, like 5 + 88, or "Hi!". Remember that all statements inside functions must have a semicolon at the end.

Sidenote: If you want to print text to the screen, you must enclose them in double quotes. If you don't do this, you will get an error. If you want to do stuff with numbers, then put %d in double quotes, add a comma and type the calcuation you want to do in the parenthesis.
