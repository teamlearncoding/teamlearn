Data Types



C, compared to other languages has few builtin data types.
But you might wonder what are those data types? Well

  • int: stores a whole number.
  • float: stores a number with a fractional part.
  • double: float but two times more precise.
  • char: stores a character. 

                        
            #include 

            int main(){

                int integer = 1;
                float floating_point = 12.2;
                double float_times_two = 155.66;
                //Characters are surrounded
                //by single quotes ''
                char grade = 'A';

                //%d is for integers
                //%f for floats and doubles
                //%c for characters
                //\n means a new line
                printf("%d\n", integer);
                printf("%f\n", floating_point);
                printf("%f\n", float_times_two);
                printf("%c\n", grade);
            }
